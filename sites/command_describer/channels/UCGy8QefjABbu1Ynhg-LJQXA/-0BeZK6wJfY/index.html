    コマンド本体

    tag @r add ken

    clear @a[tag=ken] minecraft:iron_sword 1 1

    replaceitem entity @a[tag=ken] slot.weapon.mainhand 0 iron_sword

    execute @a[tag=ken] ~ ~1002 ~ summon boat ^ ^ ^0.4

    execute @a[tag=ken] ~ ~1000 ~ execute @e[r=3,type=boat] ~ ~ ~ tp @s ^ ^ ^ facing @p[tag=ken]

    tag @a[tag=ken] add sentaku

    execute @a[tag=sentaku,tag=zimen] ~ ~1000 ~ tag @e[r=3,type=boat] add kiru2

    execute @a[tag=sentaku,tag=!zimen] ~ ~1000 ~ tag @e[r=3,type=boat] add kiru

    tag @a remove sentaku

    tag @a remove ken

    プレイヤーが空中にいるかどうか感知するコマンド

    tag @a add zimen

    execute @a ~ ~ ~ detect ~ ~-0.1 ~ air 0 tag @s remove zimen
